Shrirang Mare, "ZEBRA: For secure and seamless device interaction," Dartmouth College

Although computing devices have become ubiquitous, users' interaction with them is not as seamless and 'invisible' as it should be in ubiquitous computing. These aspects are particularly challenging for security interfaces. We identify two main problems, which if resolved would reduce the effort required to interact with devices, making the experience more seamless for the users and would improve security on devices: i) effortless user authentication, and ii) continuous user authentication. We propose to tackle these problems using a new authentication scheme called Zero-Effort Bilateral Recurring Authentication (ZEBRA). In ZEBRA users wear a wireless wrist bracelet all the time (or at least when they interact with devices), and this bracelet assists devices in initial authentication and continuously reaffirms the identity of the device user.

Kaikai Liu and Xiaolin Li, "Guoguo: Smartphone-based High-precision Indoor Location Ecosystem and Services," University of Florida

Locating a smartphone at the centimeter-level has significant implication for potential indoor location services and applications compared with existing coarse-grained solutions. In this paper, we propose Guoguo ecosystem to fill the longlasting gap of smartphone-based indoor localization. Guoguo consists of a constellation of low-complexity anchor network that enabling fine-grained localization by leveraging modulated acoustic beacons. We propose algorithms to improve its robustness, coverage, accuracy, and refresh rate. Our prototype shows centimeter-level localization accuracy and promising for real deployment and application. Such results are expected to make smartphone indoor localization available in our daily activities.

Kartik Muralidharan, "Jarvis: A Context-Aware Adaptive Intervention Engine," Singapore Management University

Context is defined as any information that can be used to characterize the situation of an entity, where an entity can be a person, place, or physical or computational object. Context-awareness or context-aware computing is the use of this context to provide task-relevant information and/or services to a user. With the advent of smartphones, one has access to large amount of highly contextual data making possible several context aware services that weren't before. In this work I give an overview of the design of Jarvis: an experimentation platform, that helps service providers get a better understanding of the underlying user context.

Tan Kiat Wee, "(Resource) Efficient Scalable Deep Context Sensing for Mobile Systems (Apps)," Singapore Management University

This dissertation research aims to developed and build a mobile deep context collector that is not only able to sense human activity accurately, but also perform such collection in a scalable and resource-efficient manner that additionally does not disrupt the usability of different applications. This research direction is motivated both by the rapid adoption of mobile devices, especially smartphones, by consumers worldwide, and by the changing patterns of content con- sumption and creation on these devices. These changes has seen new use cases (social, financial, communication, cloud- computing, content consumption, etc) beyond the traditional communication tool of a phone. This makes mobile de- vices an attractive research domain where new insights can be learned from the user as well as new and radical system trade-offs (power, scale, user annoyance) can be achieved at the same time. In this dissertation work, I will focus on the four key areas, (resource) efficiency, scalability, minimising user disruption and building of useful apps in different do- mains that utilised parts or all of these areas.

Chungkuk Yoo, "Towards Total Recall: A Mobile Platform for Centripetal Lifelogging in Everyday Life," KAIST

Life monitoring services are emerging as promising mobile applications, pursuing to build a user's autobiographical memories. To date, initial attempts of lifelogging services have been proposed, capturing what I see, hear, meet, and visit. These empirical and environmental contexts, centrifugal contexts, may help a user reminisce about the past. On the other hand, I focus on an important key feature of lifelogging which has been unexplored so far, i.e., centripetal context. The centripetal context is about oneself including inner world, appearance, reaction to external stimulation and social image in others' eyes, etc. With centripetal lifelogging in a fine-granule and momentary manner, lifelogging services enable not only total recall of a user but also centripetal context aware services. In my thesis, I propose a full-fledged everyday mobile platform for fine-granule centripetal lifelogging in a fine-granule and momentary manner.

Huber Flores and Satish Narayana Srirama, "Adaptive Code Offloading and Resource-intensive Task Delegation for Mobile Cloud Applications," Tartu University

Mobile cloud computing is arising as a prominent domain that is seeking to bring the massive advantages of the cloud to the resource constrained smartphones, by following a delegation or offloading criteria. In a delegation model, a mobile device consumes services from multiple clouds by following their Web API. In the offloading model, a mobile application is partitioned and analyzed so that the most computational expensive operations at code level can be identifid and offladed for remote processing. This paper proposes a mobile cloud architecture that enhances the decision of offloading process with asynchronous cloud processing that targets cloud infrastructure and code offloading traces, also taking advantage of the middleware based delegation model.

Xiping Hu and Victor C.M. Leung, "Application-oriented Mobile Social Networking System for Mobile Crowdsourcing," The University of British Colombia

With mobile devices providing ubiquitous access to social network applications and services, there is a remarkable trend to utilize mobile crowdsourcing to handle a diversity of problems that involve real-time data collection and processing, and collaborations among numerous participants through their mobile devices. Beyond previous work on mobile crowdsensing, which has mainly considered the processing of diverse sensing data through mobile devices, mobile crowdsourcing may further be applied to exploit the collection of human intelligence through users' mobile devices, and the interactions between human intelligence and services of mobile devices. This not only could provide ubiquitous services to users, but also promote human collaboration. Thus, mobile crowdsourcing potentially could be applied to various application areas and bring huge economic benefits to important areas in society; e.g., in transportation it could help to improve the transportation efficiency, in disaster situations it could provide an efficient and effective means of supporting rescue operations immediately after disasters, facilitate disease report and crisis management, and so on.
